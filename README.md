# Tài liệu hướng dẫn BeagleBone Black - Hệ thống Buildroot/Yocto Gateway
Repo này bao gồm hướng dẫn chi tiết cho 5 phần:
1. Quy trình cài đặt BBB từng phần:
    - Chuẩn bị Cross Toolchain
    - Biên dịch và cài đặt Uboot
    - Biên dịch và cài đặt Kernel
    - Cài đặt System Files (Root Filesystem)
2. Quy trình cấu hình, cài đặt, sản phẩm của Buildroot.
    - Tạo Distro sử dụng Buildroot
    - Hướng dẫn thêm, sửa, xóa package trong Buildroot
    - Quy trình thêm 1 Package (Application, Driver vào Buildroot)
3. Quy trình cấu hình, cài đặt, sản phẩm của Yocto.
    - Tạo Distro sử dụng Yocto
    - Hướng dẫn thêm, sửa, xóa package trong Yocto
    - Quy trình thêm 1 Package (Application, Driver vào Yocto)
4. Phát triển và Tích hợp Application
    - Chuẩn bị (Cross-Toolchain) từ Buildroot
    - Biên dịch và cài đặt
    - Quản lý phụ thuộc
    - Hướng dẫn cơ bản về Makefile
    - Tích hợp vào Buildroot.
5. Phát triển và Tích hợp Device Driver
    - Chuẩn bị (Cross-Toolchain) từ Buildroot
    - Biên dịch và cài đặt
    - Quản lý phụ thuộc
    - Makefile
    - Tích hợp vào Buildroot.
